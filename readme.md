## Getting Started
### Dependencies (Only for VS Code)
* Expo (expo-cli)
* react-native-elements (no need to install)
* react-navigation (no need to install)
* popup-ui (no need to install)
### Installing
* Install expo-cli
```
npm install -g expo-cli
```

### Executing program
* How to run the program

```
expo start
```
